# vue-library-app

## 📜 Summary
Vue.js + Firebase + Element UI를 이용한 간단한 독서 기록과 목표를 쓰고, 공유하는 웹 앱.

- Figma로 프로토타입 제작
- Vue.js Vuex 사용해 중앙에서 데이터 관리
- Firebase로 인증(가입, 로그인), Firestore, Storage, Query 활용
- (예정) 반응형 웹
- (예정) 배포, 공유
